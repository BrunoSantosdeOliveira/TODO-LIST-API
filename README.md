## 👨🏻‍💻 Objetivos
<p>-Construir uma API .Net  que possua 2 endpoints principais:</p>
<p>-Recuperação de todas as contas bancárias</p>
<p>-Recuperação dos detalhes de 1 conta bancária (inclusive dados do seu titular)</p>
